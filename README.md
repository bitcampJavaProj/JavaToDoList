# JavaToDoList
Server + Client + DB 투두리스트

## 💻 프로젝트 소개
투두리스트

## 🗓 개발 기간
2024.2.19.월 ~ 2024.2.23.금

### 👥 멤버 구성
- 팀장(서혜리): 투두리스트 작성/삭제/수정
- 팀원(권재원): 투표 등록/삭제
- 팀원(김동우): 투표 조회/수정
- 팀원(박지우): 투두리스트 조회
- 팀원(유민진): 회원가입/로그인/전체 채틷

### ⚙ 개발 환경
- Java 17
- JDK 17
- IDE: Eclipse
- Database: MySQL 8.0.36
